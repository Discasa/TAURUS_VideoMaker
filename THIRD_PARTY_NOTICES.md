# Third-Party Notices

## FFmpeg

This project uses FFmpeg and FFprobe executables as a local runtime dependency.

FFmpeg is a separate third-party project. Its license depends on the exact build configuration used for the copied binaries. Check the runtime directly with:

```powershell
.\ffmpeg\bin\ffmpeg.exe -L
.\ffmpeg\bin\ffmpeg.exe -version
```

Project website: https://ffmpeg.org/
